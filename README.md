# python_dashboard1
Projeto da Faculdade - 4º Semestre - IOT E INDÚSTRIA 4.0 EM PYTHON - Profª : Joelma O projeto consiste em um Dashboard, composto de gráficos e cards com indicadores para sensores de arduíno. Para fins demonstrativos, foram utilizados dados gerados randômicamente.
1 - É necessário ter Python no seu computador foi utilizado o python 3.8 no desenvolvimento e execução do programa; 2 - Instale as bibliotecas python; 3 - Execute o programa pelo index.py; 4 - O serviço roda no link: localhost:5000 abra em qualquer navegador;
